üìã Validation Steps

1Document Identification

Sort and identify all documents received: Pallet Notification, LOSCAM, Customer Receipt, Ship Document, Invoice.

Required: Before proceeding

2Check All Documents Present

Verify all required documents are here. Missing documents means validation fails immediately.

Minimum: Ship Document + Invoice required

3Check for Pallets

Determine if this delivery includes pallets by checking for pallet-related documents.

Determines validation path (with or without pallets)

4Check Pallet Documents

If pallets are present: Verify warehouse marks, delivery signatures, pallet counts, and damage notes.

Only checked if pallets are included in delivery

5Check Ship Document

Always checked: Verify dispatch stamps, time signatures, carrier information, and delivery timestamps.

Always Required: Every delivery must have valid ship document

6Check Invoice

Verify PO numbers, case counts, item descriptions, and pricing match delivery documents.

Always Required

7Final Cross-Check

Verify all documents match each other: PO numbers align, case counts match, signatures are consistent across all documents.

Final check before completion
